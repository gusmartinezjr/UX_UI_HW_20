Introduction

The Challenge

The original non-profit website had a very confusing and an overall unorganized website. Both the donation and volunteering did not have a call to action to inform visitors. We felt that as a team we could solve this problems.

The Solution

The goal of this project was to create a responsive website for the 'Voices of Children' organization to help improve donor and volunteer support, clearly communicate the mission statement, and show the community impact.
